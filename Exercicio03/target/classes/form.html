A plataforma iLibrary é um Web Service desenvolvido para a disciplina Trabalho Interdisciplinar II: Bancos de Dados e Sistemas Inteligentes, do curso de Ciência da Computação, da Pontifícia Universidade Católica de Minas Gerais. Esta página permite que os usuários consultem, adicionem, atualizem ou removam itens de um Banco de Dados simulado, proporcionando uma experiência interativa e imersiva. Através do uso de tecnologias web modernas, como HTML, CSS, JavaScript, Java, Eclipse, Bootstrap e PostgreSQL, a página é simples de usar e de entender.
